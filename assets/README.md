# Assets 文件夹

这个文件夹用于存放项目的静态资源文件。

## 📸 待添加的图片

请将你的项目海报图片放在这里：

- **文件名**: `framedetective-banner.jpg`
- **格式**: JPG 或 PNG
- **推荐尺寸**: 1920x600 或 1920x1080 (16:9)
- **文件大小**: < 500KB

## 🎨 海报图片来源

你之前分享的那张海报图片需要：

1. 保存到本地
2. 重命名为 `framedetective-banner.jpg`
3. 放到这个文件夹 (d:\sammodel\assets\)
4. 提交到 Git

## 📝 使用方法

图片添加后，在 README.md 中会自动显示：

```markdown
<img src="assets/framedetective-banner.jpg" alt="FrameDetective Banner" width="100%">
```

1. 在 GitHub 上创建新仓库
   https://github.com/new

2. 添加远程仓库（替换为你的仓库地址）：
   git remote add origin https://github.com/你的用户名/sammodel.git

3. 添加所有文件：
   git add .

4. 提交代码：
   git commit -m "Initial commit: SAM3 Video Analysis System"

5. 推送到 GitHub：
   git branch -M main
   git push -u origin main

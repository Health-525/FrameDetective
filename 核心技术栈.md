# FrameDetective 核心技术栈 (Top 6)

## 🎯 最主要的 6 个技术

### 1. **Node.js** (>= 18.0.0)
- **类型**: JavaScript 运行时环境
- **作用**: 后端服务器基础
- **为什么重要**: 整个后端 API 的运行基础，支持异步 I/O 和高并发处理

### 2. **Express.js** (^5.2.1)
- **类型**: Web 应用框架
- **作用**: HTTP 服务器和 API 路由
- **为什么重要**: 提供 RESTful API 接口，处理所有 HTTP 请求

### 3. **SAM3 (Segment Anything Model 3)**
- **类型**: AI 视频分割模型
- **提供商**: Replicate
- **作用**: 视频中的对象检测与分割
- **为什么重要**: 项目的核心 AI 能力，实现智能视频分析

### 4. **React** (^19.2.0)
- **类型**: 前端 UI 框架
- **作用**: 构建用户界面
- **为什么重要**: 提供反人口贩卖系统的交互界面

### 5. **Sharp** (^0.34.5)
- **类型**: 图像处理库
- **作用**: 图像格式转换、分析、Base64 编码
- **为什么重要**: 高性能处理视频帧和图像数据

### 6. **Replicate SDK** (^1.4.0)
- **类型**: AI 模型 API 客户端
- **作用**: 调用 SAM3 和其他 AI 模型
- **为什么重要**: 连接 AI 模型的桥梁，实现视频分析功能

---

## 📊 核心技术栈总结

| 技术 | 版本 | 层级 | 核心作用 |
|------|------|------|----------|
| Node.js | >= 18.0.0 | 运行时 | 后端基础环境 |
| Express.js | ^5.2.1 | 框架 | API 服务器 |
| SAM3 | Latest | AI 模型 | 视频对象分割 |
| React | ^19.2.0 | 前端框架 | 用户界面 |
| Sharp | ^0.34.5 | 图像处理 | 帧处理与分析 |
| Replicate SDK | ^1.4.0 | AI 集成 | 模型调用接口 |
